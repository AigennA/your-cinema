## ğŸ¨ Key Features Explained

### Movie Recommendation System
The website includes an intelligent movie recommendation assistant that:
- Asks about your current mood (happy, adventurous, relaxed, scared, thoughtful)
- Inquires about your viewing situation (alone, date, family, friends)
- Suggests the best movies from the current lineup based on your preferences
- Highlights the top recommendation with a special badge

### Responsive Navigation
- Desktop: Full horizontal navigation with social media icons
- Mobile: Hamburger menu with slide-out navigation panel
- Smooth scroll navigation to different sections

### Movie Card Design
Each movie features:
- High-quality poster image
- Age rating badge
- Play trailer button (overlay)
- Movie information (title, genre, duration)
- Star rating
- Available showtimes with booking functionality
